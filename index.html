Where@bout

A social media application.

Where@bout is a web application for users to share their favorite photos of places they've been. Rather than traditional social media apps that are user-focused, Where@bout focuses on the place of interest. The benefits are two fold: users can follow what's happening at their favorite places, and see a real-time snapshot of the place of interest. Businesses benefit from this application by having customer-generated marketing.
Some of the features include the following:
Users can...

  • "Follow" places of interest
  • "Like" other users' photos
  • "Comment" on photos
  • "Discover" new places
Other features:
  • Secure log-in
  • Reset password
  • Login with facebook
  • Full profile creation
  • Uploading photos from webcam or saved files
The technologies used were:
the MEAN stack (MongoDB, Express, AngularJS, and Node.js), as well as JavaScript and UI Bootstrap. Integration with:
  • Google Maps API (for geolocation)
  • Facebook API (for user authentication)
Other technologies:
  • Cloudinary (for cloud photo storage)
  • MongoLab (for Administrative database)

Live demo here:both on web and mobile whereaboutapp.herokuapp.com, and you can view the project in github github.
